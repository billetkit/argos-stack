billetkit-voice-grader

catch AI-generated slop before it ships

problem: your AI agent writes "delve into", "leverage", "I hope this helps" — instant credibility loss

billetkit-voice-grader is an MCP server that scores text for AI tells. run it before publishing social posts, documentation, customer emails. get a 0-100 score + specific flags.

install (free, MIT license)

pip install billetkit-voice-grader

add to your MCP config:

{
  "mcpServers": {
    "billetkit-voice-grader": {
      "command": "python",
      "args": ["-m", "billetkit_voice_grader"]
    }
  }
}

what it catches

  • anti-tell wordlist: delve, leverage, tapestry, furthermore, holistic, synergy, empower, paradigm, etc.
  • filler phrases: "I hope this helps", "let me know if", "feel free to", "happy to help"
  • overused intensifiers: "very", "really", "absolutely", "incredibly"
  • corporate slop: "circle back", "touch base", "synergize", "low-hanging fruit"
  • hedging: "I think", "perhaps", "it seems", "arguably"
  • repetitive structure: same sentence openers, predictable rhythm

returns JSON with score (0-100, higher = better) and specific violations:

{
  "score": 42,
  "violations": [
    {"term": "delve into", "category": "anti-tell"},
    {"term": "furthermore", "category": "anti-tell"},
    {"phrase": "I hope this helps", "category": "filler"}
  ],
  "pass": false
}
